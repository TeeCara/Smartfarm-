# Smartfarm-
1. 양액 A제

사용 방법

양액 탱크 또는 자동 양액기에서 A제 전용 탱크에 따로 희석함

물에 먼저 희석한 뒤, 설정된 비율로 관수 라인에 주입

B제와 절대 원액 상태에서 섞지 않음


포함 성분(대표적)

칼슘(Ca)

질산태 질소(NO₃⁻)


사용하는 이유

칼슘은 세포벽을 튼튼하게 만들어 작물의 조직을 강화함

생장점 발달, 잎과 줄기 생육에 필수

B제 성분과 바로 섞으면 침전(굳음) 이 생기므로 분리 사용 필요



---

2. 양액 B제

사용 방법

A제와 마찬가지로 B제 전용 탱크에 따로 희석

자동 제어 시스템이 A제와 함께 정해진 비율로 동시에 주입


포함 성분(대표적)

인(P), 칼륨(K)

마그네슘(Mg)

미량원소(철, 망간, 아연 등)


사용하는 이유

인: 뿌리 발달과 에너지 전달에 중요

칼륨: 광합성, 당 이동, 품질 향상

미량원소: 효소 작용과 생리 대사에 필수

A제와 분리함으로써 영양 균형과 용해 안정성 유지



---

3. 물 소독약

사용 방법

관수용 물이나 양액 원수에 소량을 정량 투입

정기적으로 또는 병원균 위험이 있을 때 사용

작물에 직접 닿기 전, 물 단계에서 처리


종류 예시

차아염소산

과산화수소

오존수 등


사용하는 이유

물속의 세균, 곰팡이, 바이러스 제거

배관 내부의 바이오필름(미생물 막) 방지

병 발생 감소 → 농약 사용 줄임

작물 생육 환경을 위생적으로 유지



---

4. 스마트팜에서 이렇게 나누어 쓰는 이유 (정리)

구분	이유

A제 / B제 분리	침전 방지, 영양소 손실 예방
자동 희석·주입	작물 생육 단계별 정밀 관리
물 소독	병해 예방, 수경 시스템 안정화
정량 사용	작물 피해 방지, 품질 균일화



---

한 문장 정리

> 스마트팜에서는 작물에 필요한 영양을 안정적으로 공급하고 병해를 예방하기 위해 양액 A제와 B제를 분리 사용하고, 물 소독약으로 관수 환경을 위생적으로 관리한다.
